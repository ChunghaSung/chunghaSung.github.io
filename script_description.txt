Installations.
- https://jekyllrb.com/docs/installation/ubuntu/

## After clone the repo
-`bundle install`
-`npm install`
- All should be installed all packages correctly to re-build the site

## To update CSS
- To run gulp locally, 
    type `bundle exec gulp [command]`
- For commands, please check ``gulpfile.js`` file.
- Example
    `bundle exec gulp jekyll-build` to build jekyll site

## To run the site with jekyll
- bundle exec jekyll serve
